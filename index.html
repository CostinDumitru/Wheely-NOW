Documentation

Wheely NOW is a streamlined web application designed to simplify the process of ordering and managing wheelchair orders for a trained technician in this field. The platform offers a user-friendly interface where users can quickly fill out order forms and review submitted orders related to mobility support.

The application's visual identity is built around a modern and accessible blue theme, using the primary color #17AFFF. This color was selected to convey trust, clarity, and friendliness — all qualities essential in healthcare-related platforms. It is consistently applied across the header and navigation buttons to maintain visual cohesion and guide user interaction.

Navigation is simplified through three core sections:

  • Documentation – usage guides, technologies used and support info.
  • Order Form – a quick-access form to submit new wheelchair order requests.
  • Orders – a section to view, edit or delete all previous submissions.
